# Base Library

This is the **base library**. It is the stable, approved version of the code.

## Current Version

See [VERSION](./VERSION) for the current version number.

## How Changes Get Here

Changes are promoted from the `mod/` directory using the
[Mod → Base promotion workflow](../.github/workflows/mod-to-base.yml).
A human must manually trigger the workflow and approve the resulting Pull Request
before any changes land here.
